Trash Talk — User Guide

Panduan Penggunaan Aplikasi Monitoring Sampah

Selamat Datang di Trash Talk

Trash Talk adalah sistem monitoring tong sampah berbasis IoT yang membantu Anda memantau status dan volume sampah di berbagai lokasi secara real-time.

Panduan ini akan membantu Anda memahami cara menggunakan aplikasi Trash Talk dengan efektif.

1. Cara Kerja Sistem

Sistem Trash Talk menggunakan sensor IoT untuk memantau kondisi tong sampah secara real-time.

  • Sensor Ultrasonik mengukur volume sampah dalam tong
  • Sensor GPS Neo 6mv2 berfungsi untuk menentukan posisi geografis (latitude dan longitude) tong sampah
  • Status Tong ditentukan berdasarkan kapasitas: kosong, sebagian terisi, atau penuh
  • Data dikirim ke server melalui jaringan IoT untuk diproses dan ditampilkan

2. Tampilan Dashboard

Dashboard menampilkan status semua tong sampah dalam sistem dengan indikator visual yang mudah dipahami.

  • Hijau: Tong kosong atau sedikit terisi (0-50%)
  • Kuning: Tong setengah penuh (51-80%)
  • Merah: Tong hampir penuh atau penuh (81-100%)
  • Abu-abu: Sensor tidak aktif atau error

Ikon tong sampah akan berubah sesuai dengan status volume yang terdeteksi.

3. Notifikasi & Peringatan

Sistem akan mengirimkan notifikasi ketika terjadi kondisi tertentu yang memerlukan perhatian.

  • Tong Penuh: Notifikasi dikirim ketika tong mencapai 80% kapasitas
  • Error Sensor: Peringatan ketika sensor tidak mengirim data
  • Kondisi Abnormal: Peringatan untuk kondisi tidak biasa seperti kebocoran atau kerusakan

Notifikasi akan muncul di dashboard dan dapat dikonfigurasi untuk dikirim via email atau SMS.

4. Cara Refresh Data

Data pada dashboard dapat diperbarui secara otomatis atau manual.

  • Auto-Update: Data diperbarui setiap 30 detik secara otomatis
  • Manual Refresh: Klik tombol di pojok kanan atas dashboard
  • Force Refresh: Tekan F5 atau Ctrl+R untuk memuat ulang halaman

Indikator loading akan muncul saat data sedang diperbarui.

5. Cara Menangani Error

Berikut adalah langkah-langkah untuk menangani masalah yang mungkin terjadi:

  • Sensor Tidak Terbaca: Periksa koneksi power dan jaringan sensor
  • Koneksi Internet: Pastikan perangkat terhubung ke internet
  • Data Tidak Up-to-Date: Lakukan manual refresh atau periksa status server
  • Tong Selalu Menunjukkan Penuh: Periksa apakah ada penghalang di depan sensor

Jika masalah berlanjut, hubungi tim teknis untuk bantuan lebih lanjut.

6. FAQ (Pertanyaan Umum)

Seberapa akurat pengukuran volume sampah?

Sensor ultrasonik memiliki akurasi ±3% dalam kondisi normal. Faktor seperti bentuk sampah dan kelembaban dapat mempengaruhi akurasi.

Berapa lama baterai sensor bertahan?

Dengan pengaturan default, baterai lithium dapat bertahan 6-12 bulan tergantung frekuensi pengiriman data.

Bagaimana cara menambahkan tong sampah baru ke sistem?

Hubungi administrator untuk mendaftarkan perangkat sensor baru. Setelah terdaftar, tong akan muncul di dashboard secara otomatis.

Apakah data saya aman?

Ya, semua data dienkripsi selama transmisi dan disimpan di server yang aman dengan akses terbatas.
